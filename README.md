# Theme Park Tycoon

A Theme park tycoon game, built with static HTML, CSS and JS. Its a very fun game as well!

## The game!
Build and manage a small theme park by placing paths and attractions, guiding guests through ticket booths, and growing your money over time! Expand the theme park, and built how you want!

## Current Features
- Grid based building system
- Path-based guest movement with simple routing
- Left-side parking lot with animated cars and group arrivals
- Ticket booth flow at park entry (guests queue and buy tickets)
- Ride queues and ride cycles
- Direct cash from ticket sales and the rides
- Building upgrades and bulldoze option
- Expand land upgrade
- Some floating text feedback for spending and earnings

## Rides and Buildings

- Path
- Burger Stall
- Sky Swinger
- Teacups
- Ferris Wheel
- Drop Tower
- Ticket Booths (the entry points)

## How To Play

1. Click "enter the park" 
2. Build paths first so guests can move around.
3. Place attractions next to paths.
4. Keep an eye on queues, ticket prices, and upgrade costs.
5. Expand the park when you have enough money!
6. Have fun and make more money!

## Controls
- Mouse drag: move camera
- Click toolbar buttons: select build tool
- Click building: open building panel
- Enter: start the park from start screen
